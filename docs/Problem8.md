## Problem statement and definition

Your data file contains information about several people - it includes their first name, last name, age, and career. Take a look at the data file to understand the structure.

Create a Person class that stores *at least* this information about people
In Problem8.java, read the data file and save all of the people in the data file into an ArrayList of Person objects.
Print out all of the people in the list to confirm you read it in correctly
Calculate the average age of all people in the list and print it
Count how many people in the list are under the age of 25 and print them all out
Create, in code, two more people and add them to the end of the list
Bonus: can you figure out how to print the two new people back into the file? Hint: there is a class called PrintWriter that lets you write to a file using the println method! Search it up!
