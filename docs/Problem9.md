## Problem statement and definition

A pet store sells, among other things, cats. Create a Cat class with the following attributes:
Name - String
Weight (kg) - double
Age (months) - int
Cost (JOD) - double
Make sure there is a default constructor with reasonable default values AND a constructor that allows setting all attributes.
Read the data file into an ArrayList of Cat objects. Note that the data file is separated by commas, not spaces. (You can use Integer.parseInt() and Double.parseDouble() to change strings into ints and doubles respectively)
Print out all the cats names
Print "The third cat is named ______" (fill in the blank)
Six months have passed since this data file was created, update all cats appropriately.
The last cat has gained 5 kg. Update the weight on the object. Print the new weight.
The cat named Rascal died. Find that cat and remove it from the list. 
A new kitten was brought in (Angel, 3.6, 12, 25.990).  Insert it into the 2nd cell.
A new geriatric cat was found (Gimpy, 8.3, 120,  24.990). Put him on the list. 
Print the updated list with a for-each loop
Replace the 3rd cat with (Sugar, 11.6, 84, 33.25). Put the removed cat at the end of the list. 
Switch the 2nd and 4th cats.
Print the updated list names.
Remove all cats under JOD40. Print the costs of each cat remaining on the list.
All cats heavier than 8 kilos need to go on a diet - Print the names of the cats being put on a diet.
